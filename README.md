# mesh-ky.org
Public site for Kentucky LoraWAN related content.  

The intent of this site is a simple info and awareness page to unify and highlight related mesh efforts across the commonwealth.  


## Overview 
This content is intended to be rendered using [MkDocs](https://github.com/mkdocs/mkdocs). It also uses [Material](https://github.com/squidfunk/mkdocs-material) for MkDocs.

Here is the general sequence to 'serve' it locally:
1. Open your terminal. Navigate to the folder where you created the mkdocs.yml file.  
   ```cd path/to/mesh-ky.org-main```  
2. Install the tools. Use the requirments.txt file.  
   ```pip install -r requirements.txt```  
3. Start the preview server.  
   ```mkdocs serve```  
4. View the site. You will see output that looks like this:  
   ```INFO - Serving on http://127.0.0.1:8000```  
Open http://127.0.0.1:8000 in your browser.
